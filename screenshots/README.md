# Screenshots

## How to Add Screenshots

1. Take screenshots of your app on your phone:
   - Login screen
   - Home/Browse screen
   - Service detail screen
   - Chat screen
   - Profile screen

2. Name them:
   - `01-login.png`
   - `02-home.png`
   - `03-service-detail.png`
   - `04-chat.png`
   - `05-profile.png`

3. Place them in this folder

4. They will automatically appear in the documentation
