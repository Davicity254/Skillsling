## How to Change Theme

1. Go to **Profile** screen
2. Tap **"Change Theme"** button
3. Select your preferred theme
4. Theme applies instantly
5. Theme persists after app restart

## Theme Features

- **Consistent Colors**: All UI elements adapt to theme
- **Smooth Transitions**: Instant theme switching
- **Persistent**: Saves your preference
- **Accessible**: High contrast options available
- **Battery Friendly**: Dark themes save battery on OLED screens
